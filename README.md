# saddest-code-editor
The *Saddest* Code Editor `v0.0.1`
<br>A code editor written on Node.JS, ended - 08.03.2021
<br>
# Instructions
<br>
1. Download Node.js from <a href="//nodejs.org">official site</a>.<br>2. <b>Fork</b> the project.<br>3. 0pen the file explorer and in your Downloads find saddest-code-editor-main.zip file, extract it.<br>4. Open the folder (saddest-code-editor-main) and write these commands: <code>npm i</code>, <code>node index.js</code><br>5. Open ur browser and go to <a href="http://localhost:3000">http://localhost:3000</a>.
